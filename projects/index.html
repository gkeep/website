About
A system monitoring tool written in Go.
go-sys-mon supports monitoring of CPU, RAM, disks, network and temperatures.
Built as a college graduation project, go-sys-mon monitors your computer on multiple fronts: CPU, RAM, disk activiry and status, temperatures.

Program features two UI modes: terminal and web.

Terminal UI mode is helpful when you have physical or remote console access to the machine you're working on.
Web UI works best when the HTTP port is open on the machine/router. This way you can access the web interface directly from the Internet (for example from your phone or laptop outside of network the machine is on).
